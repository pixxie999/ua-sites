🤖 AI 콘텐츠 정책

전국축제정보는 행사 요약·추천 글 작성에 AI(Claude by Anthropic)를 보조 도구로 활용합니다. AI가 생성한 초안은 편집팀이 검토·수정 후 게재합니다.

✅

1차 데이터 기반: 모든 행사 정보는 한국관광공사 공식 API에서 수집한 데이터를 기반으로 합니다.

✅

편집팀 검수: AI가 생성한 요약·추천 글은 편집팀이 사실 관계를 확인하고 검수합니다.

✅

투명한 공개: AI 사용 여부를 숨기지 않으며, 각 페이지 하단에 검수일을 표기합니다.

⚠️

정보 변경 주의: 행사 정보는 주최 측 사정에 따라 변경될 수 있습니다. 방문 전 반드시 공식 홈페이지에서 확인하세요.

📊 데이터 출처

• 행사 정보: 한국관광공사 Tour API (공공데이터포털 제공)
• 관광지 정보: 한국관광공사 Tour API (관광지 데이터)
• 날씨 정보: 기상청 단기예보 API (공공데이터포털 제공)
• 후기 정보: 네이버 카페 검색 API

본 서비스는 공공데이터포털(data.go.kr)의 공공 API를 활용하며, 데이터의 정확성은 원출처 기관에 따릅니다.
